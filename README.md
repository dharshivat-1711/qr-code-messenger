# QR Code Messenger

QR Code Messenger is a simple website-based application that allows users to generate and decode QR codes for message sharing.

## Features
- Generate QR codes from text messages
- Decode QR codes back to readable text
- Clean and user-friendly interface

## Tech Stack
- HTML
- CSS
- JavaScript

## How to Run
1. Clone or download the repository
2. Open index.html in a web browser
3. Enter a message and generate QR code

## Purpose
This project was developed as an academic exercise to understand QR code technology and frontend web development.
